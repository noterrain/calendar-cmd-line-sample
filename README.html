Command Line Calendar API Sample Instructions

A starter application has been generated for you.

  1. Install Python 2.
  2. Download the starter application and unzip it. It will unzip into a directory called calendar-cmd-line-sample.
  3. Download the client secrets file and overwrite the client_secrets.json file that was included in the application. The client_secrets.json file contains your applications client id and client secret and should be stored securely.
  4. Within the directory of files you just unzipped, run your starter application:

    python sample.py

Congratulations, you are up and running! At this point you will want to add calls into the Calendar API to the sample.py file. Please read the sample.py file carefully, it contains detailed information in the comments. For more information on the Calendar API Python library surface you can visit:

https://developers.google.com/resources/api-libraries/documentation/calendar/v3/python/latest/

The Calendar API has been turned on in the Cloud Console. You can see and adjust the OAuth 2.0 project settings by visiting:

https://cloud.google.com/console#/project/218746604801/apiui

Also check out the Python Client Library documentation, and get more information on the Calendar API at:

https://developers.google.com/google-apps/calendar/firstapp
